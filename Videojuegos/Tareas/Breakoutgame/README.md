## Descripción del Proyecto
Este archivo tiene el propósito de explicar y entender el funcionamiento de mi versión modificada del clásico juego **Breakout**. 

El objetivo principal es romper la mayor cantidad de bloques posibles para superar tu propio récord. El juego no tiene un fin determinado, pero cuentas con **tres vidas**; si las pierdes todas, la partida termina.

## Mecánicas de Juego
Debes controlar una paleta horizontal para hacer rebotar la pelota, evitando que caiga al vacío mientras destruyes los bloques en pantalla.

### Controles
* **Movimiento:** Teclas `A` / `D` o las **flechas** izquierda y derecha.
* **Saque / Reiniciar:** Tecla `Espacio` (se usa tanto para lanzar la pelota al inicio como para reiniciar tras perder).

---

## Mi Aportación (Giro de Mecánica)
La característica única de esta versión es su dificultad dinámica: 

> Cada vez que logres limpiar todos los bloques de la pantalla, **el escenario girará 180 grados e invertirá los controles**. 

Este cambio busca transformar la experiencia convencional en un reto mucho más difícil y cambiante para el jugador.
